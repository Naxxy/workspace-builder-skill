- `../../_config/voice.md` — Voice and register conventions. All written output must conform to this guide.

---

## Process

Read the synthesis and both reference files before writing.

Write the final report following the structure in `references/report-structure.md`. Every claim must draw from the synthesis — do not introduce new material at this stage. If the synthesis is missing something the structure requires, flag it rather than inventing content.

Every claim in the report must be traceable: a reader who hasn't seen earlier stages should be able to identify which source supports each finding from the inline citations.

The report must stand alone as a complete document — readable and trustworthy without access to the earlier stage outputs.
